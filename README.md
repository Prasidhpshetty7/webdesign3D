# Webdesign3D

## Overview
Webdesign3D is an experimental frontend project that explores 3D-style visuals, motion, and depth in web interfaces.

## Tech Stack
- HTML
- CSS
- JavaScript

## Development Style
This project was created using **vibe coding**, focusing on creative experimentation with motion and visual depth.

## How to Run
Open the `index.html` file in your browser to experience the 3D effects.

## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run the following commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
